# TaskMate

TaskMate is an AI-powered personal assistant designed to help users efficiently manage their daily activities. 
It allows users to create and manage tasks through both text and voice input, making the experience flexible and user-friendly. 
One of TaskMate’s standout features is its ability to analyze the tone of a user’s voice to detect emotions such as happiness, stress, or sadness. 
Based on this emotional analysis, the assistant adjusts its responses to provide a more personalized and empathetic interaction. 
Users can also set intelligent reminders that notify them at the most appropriate time and track their emotional well-being through mood monitoring. 
Built using the MERN stack (MongoDB, Express.js, React, Node.js), TaskMate offers a modern, responsive, and intelligent solution that 
adapts to individual user needs and enhances everyday productivity.
